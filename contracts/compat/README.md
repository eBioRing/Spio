# spio Compatibility Matrix

**Purpose:** Record the Styio compiler ranges and protocol requirements that a given `spio` release claims to support.

**Last updated:** 2026-04-09

## Rules

- `spio` publishes after the `styio` releases it supports.
- A support entry must refer only to released compiler versions.
- Capability and contract checks still win over plain version ranges.
- This file is an implementation input, not just documentation.

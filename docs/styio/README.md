# Styio Integration Docs

**Purpose:** Hold the `styio` knowledge pack and public interface expectations required by `spio` developers.

## Scope

- what `spio` maintainers must know about `styio`
- which compiler interfaces are published
- which compiler internals must remain off-limits

## Maintenance Rule

These files document the external compiler dependency. They must not become a back door for depending on compiler internals.
